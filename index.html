📋 インポート方法

楽天RMSのRPP管理画面から以下の手順でレポートをダウンロードしてください。

  1. 集計単位：商品別を選択
  2. 集計期間：取得したい期間を指定（1ヶ月以内）
  3. 「全商品レポートダウンロード」ボタンをクリック
  4. ダウンロードしたZIPをADcheckの「＋レポート追加」からドロップ

⚙ 設定

楽天アプリID・アクセスキーを入力すると商品画像が自動取得されます。GitHub PATを入力するとデータがGitHubに保存され、どのPCからも同じデータが閲覧できます。

楽天Developers のアプリ一覧 →「アプリケーションID」欄の値です。
アプリ一覧の「アクセスキー」欄の目のアイコンをクリックすると表示されます。
楽天市場の店舗URLに含まれる英数字のコードです。
例：item.rakuten.co.jp/yukaiya/xxxxxxxx/
入力すると全PCでデータを共有できます。GitHubのSettings → Developer settings → Personal access tokens で発行（repo権限）。
